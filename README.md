## Project Overview
This project is a Power BI case study based on a public survey dataset about data-related professions
(Data Analyst, Data Scientist, Data Engineer, etc.).

The objective is to analyze career trends, salary distribution, job satisfaction, and technology usage
through an interactive dashboard.

---

## Dataset
- Public survey dataset
- Approximately 600+ respondents
- Raw data required cleaning and preparation before visualization

---

## Tools & Technologies
- Power BI
- Excel (data preparation)
- DAX (basic measures)

---

## Key Analyses
- Average salary by job role
- Job satisfaction and work-life balance
- Distribution of respondents by country
- Most commonly used programming languages
- Perceived difficulty of entering data-related roles

---

## Deliverables
- Interactive Power BI dashboard
- KPI visualizations with filtering by job role and country
- Visual summaries to support data-driven insights

---

## Note
Due to Power BI Service access limitations, the interactive dashboard is not publicly embedded.
Visual previews and analysis are available through the project portfolio.

---

## Value
This project demonstrates data preparation, visualization, and storytelling skills
to transform survey data into actionable insights.
